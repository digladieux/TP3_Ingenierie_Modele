63 // Luxury level is set in the same way as the original FORTRAN routine.
64 // level 0 (p=24): equivalent to the original RCARRY of Marsaglia
65 // and Zaman, very long period, but fails many tests.
66 // level 1 (p=48): considerable improvement in quality over level 0,
67 // now passes the gap test, but still fails spectral test.
68 // level 2 (p=97): passes all known tests, but theoretically still
69 // defective.
70 // level 3 (p=223): DEFAULT VALUE. Any theoretically possible
71 // correlations have very small chance of being observed.
72 // level 4 (p=389): highest possible luxury, all 24 bits chaotic.
